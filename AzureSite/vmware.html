VMWare Player Setup


VMware Player is a free application that allows you to run virtual machines. The free version is only available on Windows. If you would like to use a different operating system, VirtualBox is an alternative and we have similar directions for how to set it up.


Getting Started

  1. Download and install VMware Player

  2. Download the image of Kali Linux

Kali Linux is a distribution of Linux that has a lot of security related software pre-installed so that you get certain applications like HTTP sniffers wihtout any extra installations. The download is about 2.5 GB and is about an additional 11 GB after being extracted because of the reserved space that the virtual machine requires, so make sure that you have enough space before starting. The image is in a .7z file so you will need 7-Zip to extract it.

  1. Extract the Kali image

  2. Open the image in VMware Player



  1. Set up a bridged network connection so that your host computer and virtual machine will appear to have two different IP addresses from within the virtual machine.
