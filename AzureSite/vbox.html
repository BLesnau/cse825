VirtualBox Setup


VirtualBox is a free application that allows you to run virtual machines. It is available on multiple operating systems, but if you would like to use VMWare Player as an alternative, it is available for free on Windows. We have similar directions for how to set it up.


Getting Started

  1. Download and install Virtual Box

  2. Download the image of Kali Linux for Virtual Box. The password is "cse825".

Kali Linux is a distribution of Linux that has a lot of security related software pre-installed so that you get certain applications like HTTP sniffers wihtout any extra installations. The download is about 3.5 GB and is about an additional 18 GB after being extracted because of the reserved space that the virtual machine requires, so make sure that you have enough space before starting.

  1. Open the image in VirtualBox



  1. Set up a bridged network connection so that your host computer and virtual machine will appear to have two different IP addresses from within the virtual machine.



  1. Launch the virtual machine

You may get an error about not being able to find the network adapter. This is because it's looking for the network adapter of the computer that created the image. You can resolve this issue by choosing to "Change Network Settings" and then picking your adapter.
